# 📊 Global Superstore Dashboard (Power BI)

## 📌 Project Overview

This project consists of **two interactive Power BI dashboards** built using the Global Superstore dataset. The dashboards provide detailed insights into sales performance, profitability, customer behavior, and business trends.

---

## 🎯 Objectives

* Analyze sales and profit performance
* Identify high and low performing regions
* Understand customer behavior and segmentation
* Track business growth over time

---

## 📊 Dashboards Included

### 🔹 Dashboard 1: Sales & Profit Analysis

This dashboard focuses on overall business performance.

**Key Features:**

* 📈 Total Sales, Profit & Orders Overview
* 🌍 Region-wise Sales Performance
* 🛒 Category & Sub-category Analysis
* 📅 Monthly / Yearly Trends
* 🔝 Top-performing products

---

### 🔹 Dashboard 2: Customer Analysis / RFM Insights

This dashboard focuses on customer behavior and segmentation.

**Key Features:**

* 👥 Customer Segmentation (RFM Analysis)
* 💰 High-value vs Low-value customers
* 🔁 Purchase Frequency Analysis
* 📊 Customer distribution insights
* 🎯 Target customer groups

---

## 📂 Dataset

* Source: Global Superstore dataset
* Data includes:

  * Orders
  * Customers
  * Products
  * Sales & Profit

---

## 🛠 Tools & Technologies

* Power BI
* Data Cleaning & Transformation
* DAX (for calculations like RFM)
* Data Visualization
* RFM
