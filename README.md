# 📊 Global Superstore Dashboard (Power BI)

## 📌 Project Overview

This project includes **two interactive Power BI dashboards** built using the Global Superstore dataset. The dashboards provide deep insights into **business performance and customer behavior** using advanced analytics and visualization techniques.

---

## 🎯 Objectives

* Analyze sales, revenue, and profit trends
* Track year-on-year business performance
* Identify top-performing categories and regions
* Segment customers using **RFM Analysis**
* Support data-driven decision making

---

## 📊 Dashboards Included

### 🔹 Dashboard 1: Performance Tracking

This dashboard focuses on overall business performance and KPIs.

**Key Highlights:**

* 💰 Total Revenue: **1.71M+**
* 📦 Total Shipping Cost: **272K+**
* 📈 Total Profit: **289K+**
* 👥 Total Customers: **800**
* 📊 Year-on-Year Sales & Profit Growth (2012–2015)
* 🛒 Category-wise Revenue Distribution (Technology, Furniture, Office Supplies)
* 📅 Category-wise Yearly Profit Analysis
* 🌍 Region, Segment, Ship Mode filters
* 📋 Detailed transaction-level data table

---

### 🔹 Dashboard 2: Customer Segmentation (RFM Analysis)

This dashboard focuses on customer behavior and segmentation using RFM metrics.

**Key Highlights:**

* 💰 Total Sales: **1.71M+**
* ⏱ Avg Recency: **575**
* 🔁 Avg Frequency: **1.08**
* 💵 Avg Monetary Value: **2.14K**
* 👥 Customer Segments:

  * Champion
  * Loyal Customers
  * At Risk
  * Potential Loyalists
* 📊 Sales distribution by customer segments and category
* 📅 Monthly revenue trend with customer count
* 📉 Customer distribution across segments
* 📋 Detailed customer-level RFM table

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in **Power BI Desktop**
3. Use filters (Region, Category, Segment, Year, Ship Mode)
4. Navigate between dashboards using tabs

---

## 💡 Key Insights

* 📈 Sales and profit show consistent growth over the years
* 💻 Technology category generates the highest revenue
* ⚠ Some categories have high sales but lower profit margins
* 👑 "Champion" customers contribute significantly to revenue
* 🔻 "At Risk" customers need targeted retention strategies
* 📅 Monthly trends highlight seasonal variations in sales
